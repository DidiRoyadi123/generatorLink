Cara Membuat User ID Pengguna Aplikasi MPD

Tanggal Terbit: 2017-06-15T19:02:00.000-07:00

Sahabat SMK, kali ini saya akan melanjutkan membuat torial tentang Aplikasi MPD yaitu mengenai Cara Cara Membuat User ID Pengguna Aplikasi MPD.

Pada Artikel sebelumnya saya telah menjelaskan mengenai Cara Installasi Aplikasi MPD beserta updaternya dan juga Cara Mengambil Data dari Dapodik ke Aplikasi MPD.

Perlu anda ketahui, pengguna Aplikasi MPD bukan hanya Admin saja. Melainkan juga mencakup Guru, Wali Kelas, BK dan Siswa.



Perhatikan saja di bagian fitur “Level User” pada halaman Login Aplikasi MPD. Di situ ada beberapa pihan yaitu, Guru, Admin, Wali Kelas, Guru BK dan siswa

Untuk itu mereka harus dibuatkan User ID terlebih dahulu agar bisa melakukan login. Jika anda langsung memilih level user User tanpa membuatkannya User ID dahulu maka dapat dipastikan tidak akan bisa masuk ke dalam Aplikasi MPD.

Lalu bagaimana Cara Membuat User ID Pengguna Aplikasi MPD?

Untuk membuat User ID pengguna Aplikasi MPD harus dilakukan oleh Admin. Adapun caranya adalah sebagai berikut:

  • Buka Aplikasi MPD SMK di browser Anda
  • Untuk mengatur semua Level User maka anda harus masuk sebagai Admin terlebih dahulu, maka dari itu silahkan Login sebagai Admin
  • Setelah itu anda akan masuk ke Menu awal Aplikasi MPD
  • Klik Menu “Data User” yang berada di sebelah kiri. Letaknya berada di bawah Menu Ambil Data Dapodik dan Data Referensi Dapodik
  • Kemudian nanti akan tampil Laman Data User, yang berisi Data User Guru, Data User Wali Kelas, Data User BK dan Data User Siswa. Karena belum diisi maka semua Data User tersebut masih kososng
  • Kemudian Isi semua Data User tersebut. Caranya ada 2, yaitu secara Manual dan Otomatis
  • Pengisian data user secara manual caranya ialah, Klik Tombol “Tambah User”, maka akan muncul beberapa data yaitu:
  1. Hak Akses isi dengan status User misalnya Guru
  2. Pilih User isi dengan data nama Guru di sekolah yang akan dibuatkan User ID
  3. User ID isi dengan kode untuk login. Penentuan user ID ini bebas yang penting anda mengingatnya
  4. Password isi dengan password yang akan anda gunakan untuk Login Aplikasi MPD
  5. Setelah semua terisi, Klik Tombol “SIMPAN” berwarna hijau di bawahnya. Maka akan muncul pemberitahuan bahwa data user berhasil ditambahkan. Dengan cara ini anda harus mengentri satu per satu data user di sekolah anda.
  • Jika ingin lebih cepat, saya anjurkan anda untuk membuatkan Data User secara Otomatis, Caranya adalah pada Menu sebelah Kanan Atas ada Menu “Generate Ulang User, silahkan anda Klik
  • Maka akan muncul pemberitahuan “Data User Guru, Wali Kelas, BK dan Siswa berhasil di generate ulan, Default Password untuk semua user adalah: 123456”
  • Setelah tombol tersebut anda klik maka semua user akan secara otomatis dibuatkan User ID oleh Aplikasi MPD
  • Bagi Guru yang juga menjabat sebagai Wali Kelas maka akan mempunyai 2 User ID
  • Biasanya untuk Username Guru adalah Nama Lengkapnya, Wali kelas adalah Nama Kelasnya, Guru BK adalah Nama Lengkap dan Siswa adalah NISN
  • Sedangkan Passwordnya semuanya sama yaitu “123456”
  • Untuk lebih jelasnya, silahkan anda Klik Tombol “Download Data User” maka semua data User ID akan diunduh dan bisa dilihat Username serta passwordnya.
