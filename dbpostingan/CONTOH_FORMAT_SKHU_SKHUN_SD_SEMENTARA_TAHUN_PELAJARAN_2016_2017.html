Bagi Anda yang membutuhkan format SKHU SKHUN SD sementara, berikut ini kami bagikan contoh format SKHU SKHUN SD sementara. Contoh format ini telah disesuaikan dengan juknis pengisian ijazah terbaru, sehingga dapat digunakan pada sekolah anda. Kami sediakan 2 format SKHU, yakni format SKHU untuk sekolah pelaksana kurikulum KTSP dan format SKHU untuk sekolah pelaksana kurikulum 2013. Format ini semua berekstensikan excel, jadi sangat mudah buat anda sesuaikan. Adapun sistem kerjanya adalah anda hanya menginput data pada tab sheet isian input data, mulai dari nama lengkap, NISN, Nomor Peserta Ujian, Nilai Rapor dan Ujian Sekolah. Bagi anda yang menginginkan format tersebut, silahkan unduh pada tautan berikut ini:
contoh format SKHU SD KTSP tahun pelajaran 2016/2017
contoh format SKHU SD kurikulum 13 tahun pelajaran 2016/2017
